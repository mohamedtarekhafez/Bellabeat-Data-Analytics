Bellabeat Fitness Data Analysis (Case Study)

# Project Overview
This project is a data analysis case study focused on *Bellabeat*, a high-tech manufacturer of health-focused products for women.
The goal is to analyze smart device fitness data (specifically from Fitbit trackers) to gain insights into how consumers utilize non-Bellabeat smart devices. 
These insights will help guide the company's future marketing strategy and unlock new growth opportunities.

# Tools Used
* *Data Cleaning & Manipulation:* SQL (SQL Server Management Studio) \ Microsoft Ecxel 
* *Data Visualization:* Tableau

# Data Source
The analysis utilizes the public dataset: *FitBit Fitness Tracker Data* (available on Kaggle), which contains personal fitness tracker data from eligible Fitbit users, including minute-level output for physical activity, heart rate, and sleep monitoring.

# Key Questions Addressed
1. What are some trends in smart device usage?
2. How could these trends apply to Bellabeat customers?
3. How can these trends help influence Bellabeat marketing strategy?

# High-Level Insights & Recommendations
* *Activity Tracking:* Analysis showed a strong correlation between daily step counts and calories burned, with peak active hours usually occurring during commuting times and post-work hours.
* *Sleep Optimization:* A significant portion of users spend time in bed before actually falling asleep; Bellabeat could introduce smart alerts to help optimize sleep schedules.
* *Marketing Strategy:* Focus marketing campaigns on product features that promote consistency, personalized reminders for daily step goals, and seamless sleep tracking tailored to women's wellness.
